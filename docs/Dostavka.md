# Введение 

**Чижик** - это платформа (веб-приложение) для доставки еды и товаров в любое время. Основгая цель продукта - сократить время на поиск нужных позиций и оформление заказа. Платформа позволяет отслеживать курьера в реальном времени, сохранить историю заказов и производить оплату разными способами.

# Примеры использования

**1. Регистрация по номеру телефона:** При нажатии на кнопку "Войти" в вводе номера, система отправляет смс для подтверждения личности
**2. Поиск нужного товара или продукта по нахванию:** При вводе названия в строку поиска, платформа отфильтровывает товары и показывает все рестораны и магазины, где есть данная позиция
**3. Добавление товара в корзину:** При нажатии на кнопку в виде "Корзины" рядом с товаром, он сразу же отправляется в корзину
**4. Выбор способа оплаты:** При нажатии на кнопку "Способ оплаты", платформа предлагает несколько вариантов: СПБ, картой или наличными
**5. Изменение времени доставки:** При оформлении заказа, высвечивается поле "Выбрать время доставки", где можно выбрать то время, к которому вам удобно получить заказ
**6. Повтор предыдущего заказа:** При нажатии на корзину, сверху высвечивается поле "Повторить заказ"
**7. Звонок курьеру:** При нажатии на профиль, если курьер доставляет заказ, высвечивается окошко "Связаться с курьером"
